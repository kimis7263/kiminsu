# 📊 AI Log Visualizer (Master & Commander Edition)

복잡한 AI 모델 학습 로그(CSV)를 엑셀이나 Jupyter Notebook 없이 즉각적으로 시각화하고 교차 비교(Cross-Validation)할 수 있는 데이터 사이언티스트 전용 Windows 유틸리티입니다.

## 1. 실행 환경
* **운영체제:** Windows 10 / Windows 11 (64-bit)
* **런타임 환경:** Python 3.9 이상 권장 (소스코드로 직접 실행 시)

## 2. 설치 절차 (종속 패키지)
동봉된 `AILogVisualizer.exe` 단일 실행 파일을 사용하실 경우 **별도의 설치 절차가 필요하지 않습니다.** 만약 소스코드(`src/main.py`)를 직접 실행하시려면 터미널(명령 프롬프트)에서 아래 명령어로 필수 종속 패키지를 설치해야 합니다.

```bash
pip install PySide6 pandas pyqtgraph numpy


## 3. 실행 방법 (단계별)
실행: AILogVisualizer.exe 파일을 더블 클릭하여 프로그램 창을 엽니다.

데이터 로드: 분석할 .csv 파일이나 폴더를 탐색기에서 드래그하여 프로그램 창에 던져 넣습니다(Drag & Drop).

그래프 추가: 좌측 목록에서 파일을 클릭하고, 그릴 Y축(예: Train_Loss)을 선택한 뒤 [➕ 선택 항목 누적 추가] 버튼을 누릅니다.

교차 비교: 다른 CSV 파일을 클릭하고 항목을 추가하면 기존 그래프 위에 데이터가 누적되어 그려집니다.

범례(Legend) 커스텀: 우측 상단에 뜬 범례 박스에 마우스 우클릭 후 드래그하여 자유롭게 이동시키거나, 텍스트를 클릭해 이름을 수정할 수 있습니다.

메모 및 내보내기: [📝 그래프 메모] 기능으로 그래프 위에 설명을 남기고, 상단의 [📷 고급 내보내기] 버튼을 눌러 PNG/SVG 이미지나 데이터 CSV로 병합 추출합니다.

미니 모드: 상단의 **[📌 미니모드]**를 누르면 바탕화면 구석에 플로팅 위젯으로 변신하여 작업 공간을 절약할 수 있습니다.

4. 관리자 권한 필요 여부
필요 없음: 일반 사용자 권한으로 파일 읽기 및 그래픽 렌더링 등 모든 기능이 정상적으로 동작합니다. 별도의 관리자 권한 실행(관리자 권한으로 실행)이 요구되지 않습니다.
